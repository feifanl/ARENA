# ARENA 3.0 AI Safety Curriculum

My copy of the ARENA AI Safety Curriculum from London Initiative for Safe AI (LISA). ARENA is a structured curriculum that covers deep learning fundamentals, transformers, mechanistic interpretability, reinforcement learning, LLM evaluations, and other alignment topics such as emergent misalignment. This repo tracks my progress and key takeaways.

Huge shout out to LISA for making the curriculum public and doing so much for the AI safety space!
